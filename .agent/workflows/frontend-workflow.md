# 프론트엔드 중심 워크플로우

이 워크플로우는 설레연 데이팅 앱의 프론트엔드 구현에 중점을 둔 흐름입니다.

## 1. 회원가입 및 로그인 흐름

```
앱 시작 → 로그인 여부 확인
├── YES → 메인 화면으로 이동
└── NO → 약관 동의 화면
         ├── 전체 동의 (필수/선택)
         └── 휴대폰 번호 입력 → 인증번호 확인
              ├── 기존 회원 → 로그인 처리
              └── 신규 회원 → 온보딩 시작
```

## 2. 온보딩 프로필 설정

다음 정보를 순차적으로 입력받습니다:

### 필수 정보
- **이름**: 실명 입력
- **성별**: 남성/여성 선택
- **생년월일**: 날짜 선택
- **키**: 숫자 입력
- **대학교**: 학교 검색 및 선택
- **학생증 인증**: 카메라로 촬영 또는 갤러리에서 선택
- **학과/전공**: 텍스트 입력
- **MBTI**: 16가지 중 선택
- **지역**: 위치 선택
- **프로필 사진**: 최소 1장, 최대 6장

### 선택 정보
- **관심사** (취미, 음식 등)
- **자기소개**

## 3. 메인 5개 탭 화면

### 3.1 홈/매칭 (MatchingScreen)
- 오늘의 추천 프로필 카드
- 매력 지수 표시
- Like/Pass 스와이프
- 상세 프로필 보기

### 3.2 채팅 (ChatScreen)
- 1:1 채팅
- 3:3 그룹 채팅
- AI 어시스턴스 (대화 주제 추천)
- 블록 리스트

### 3.3 커뮤니티 (CommunityScreen)
- 소소한 질문 게시판
- 연애 상담 게시판
- Block 관리 (글/댓글 차단)
- 신고 기능

### 3.4 마이페이지 (MyPageScreen)
- 프로필 수정
- 받은 매력 포인트 확인
- 매칭 히스토리
- 구독 상태

### 3.5 더보기 (MoreScreen)
- 라이프 스타일 설정
  - 음주: 안 마심/가끔/자주
  - 흡연: 비흡연/가끔/흡연
  - 연하: 싫음/괜찮음/연하만
  - 아이: 없음/미래에/상관없음
- 구독 관리
- 고객센터
- 정책/로그아웃

## 4. 매칭 시스템 로직

```
프로필 카드 표시
├── Like 스와이프 → 좋아요 전송
│   ├── 상대도 Like → 매칭 성공! 🎉
│   │   └── 채팅방 생성
│   └── 상대 아직 없음 → 대기
├── Pass 스와이프 → 다음 카드
└── Super Like → 우선 표시 + 알림
```
