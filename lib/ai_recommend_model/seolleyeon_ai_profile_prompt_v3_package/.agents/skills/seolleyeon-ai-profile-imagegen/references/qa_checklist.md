# Seolleyeon AI Image QA Checklist

Reject or mark needs_review if:
- appears under 20
- school uniform
- idol / celebrity / influencer styling
- glamour studio lighting
- sexualized pose or revealing outfit
- nightclub, party, neon scene
- distorted face, hands, legs, fingers
- unreadable or mismatched shot type
- visible logo, text, watermark, school name
- not realistic smartphone profile photo
- too synthetic, plastic skin, heavy beauty filter
- same identity's 3 shots do not look like the same person
